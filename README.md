## Quickstart

Clone the project in your development environment

```shell script
git clone https://github.com/leivio/work-at-boticario.git
```


```shell script
cd work-at-boticario
pip install psycopg2 
pip install tweepy
```

Deploy scripts dag_* in dag folder Airflow
